== Description ==

MM Duplicate is a plugin that allows you to duplicate your posts and pages. 
The mm-duplicate plugin creates new post/page with the content of original post/page. 
When duplicating a Post it will also copy all the categories and tags of that post.


The following can be achieved with mm-duplicate plugin.

1. Duplicating a post to a new post.
1. Duplicating a post to a new page.
1. Duplicating a page to a new page.
1. Duplicating a page to a new post.

You can find more information how to duplicate a post/page here http://plugins.motionmill.com/mm-duplicate.

Checkout our other plugins at http://plugins.motionmill.com

== Installation ==

1. Download mm-duplicate.zip
1. Unzip
1. Upload mm-duplicate directory to your /wp-content/plugins directory
1. Go to plugin management page and activate the plugin.
